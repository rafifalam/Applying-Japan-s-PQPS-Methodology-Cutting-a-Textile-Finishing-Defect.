# Applying-Japan-s-PQPS-Methodology-Cutting-a-Textile-Finishing-Defect.
Mentored a QC Circle using Japan's AOTS PQPS methodology — root-cause analysis, Pareto ranking, and PDCA-based countermeasures — to fix a textile finishing defect. Cut defect rate from 4.7% to 1.87%, saved ~$1,510 in reprocessing, and won a national Platinum QC award.
